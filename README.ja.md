# かわいいもふもふキャラクターSVG

このリポジトリには、SVGアニメーションで作られた可愛らしいキャラクターが含まれています。キャラクターは浮遊したりスクイーズしたりする可愛らしい動きをし、瞬きやキラキラした背景のアニメーションも備えています。

## デモ
https://code4fukui.github.io/mof_svg_char/ でキャラクターのアニメーションを確認できます。

## 機能
- キャラクターの浮遊とスクイーズアニメーション
- キャラクターの瞬きアニメーション
- キャラクターの周りにキラキラした背景アニメーション

## 使い方
`index.html` を開くことで、キャラクターのアニメーションを表示できます。
